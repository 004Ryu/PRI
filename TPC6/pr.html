    Abstract

    Para este TPC foi-nos proposta a criação de uma aplicação web relativa à gestão de uma lista de filmes pertencentes a um Base de Dados em MongoDB, na qual deveria ser possível introduzir e remover filmes, assim como visualizar a informação de um dado filme.

    Foi, sem dúvida, um projeto eficiente relativamente ao aprofundar do entendimento do conceito de paths em express, agora com o acréscimo das funcionalidades relativas à utilização uma Base de Dados em MongoDB.
